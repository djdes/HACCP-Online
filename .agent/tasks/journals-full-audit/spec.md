# journals-full-audit spec

Frozen: 2026-04-12.

## Цель

Для всех 35 журналов (минус эталонный «Гигиенический журнал») проверить:
- UI на https://wesetup.ru соответствует JPG из `c:/www/Wesetup.ru/journals/<русское-название>/`.
- Все кнопки на странице журнала работают.
- Кнопка «Печать» открывает в новой вкладке `/api/journal-documents/[id]/pdf` с `Content-Type: application/pdf`, статус 200, непустое содержимое.
- PDF содержит шапку, корректную таблицу, те же строки/значения что в UI — без UUID, без ISO-дат, без mojibake.
- Введённые данные персистентны (сохраняются в БД).

## Acceptance Criteria

- **AC1** — Визуальное соответствие UI ↔ эталонному JPG (структура, колонки, кнопки).
- **AC2** — Все кнопки на странице журнала отрабатывают без ошибок.
- **AC3** — Кнопка «Печать» открывает PDF в новой вкладке; 200; корректный Content-Type.
- **AC4** — Содержимое PDF соответствует UI (шапка, таблица, значения).
- **AC5** — Данные, введённые в UI, персистят после перезагрузки.

## Политика пуша

Все коммиты копятся локально, один финальный `git push origin master` после завершения.
