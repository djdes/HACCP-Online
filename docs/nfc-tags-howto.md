# NFC-метки на оборудование (как QR, но удобнее)

## Зачем

QR-наклейки уже работают (`/equipment-fill/[id]`). Но повар в перчатках с измерительным щупом часто не может комфортно открыть камеру → навести → отсканировать.

**NFC-метка** — поднёс телефон, открылась нужная страница. Один тап вместо четырёх.

## Что нужно

- **NFC-метки**: NTAG213/215/216 — ≈3 ₽/штука на Озоне или AliExpress (искать «NTAG215 NFC tag»). На 1 ресторан = 50-100 рублей расходка.
- **Программа для записи**: бесплатная Android-приложение **NFC Tools** (Wakdev) или iOS Shortcuts (нативно).
- **Никакого кода в WeSetup не нужно** — для NFC используются те же URL `/equipment-fill/[id]` что и для QR.

## Как записать (Android, NFC Tools)

1. Установить «NFC Tools» из Google Play.
2. В админке WeSetup `/settings/equipment` найти нужное оборудование.
3. Тапнуть кнопку **«Скопировать ID»** (новая кнопка рядом с edit).
4. В NFC Tools → вкладка «Запись» → «Добавить запись» → «URL/URI».
5. Ввести: `https://wesetup.ru/equipment-fill/<id>` (вставить скопированный ID).
6. Поднести метку к задней части телефона → «Записать».
7. Наклеить на холодильник.

## Как записать (iPhone, Shortcuts)

iPhone не пишет NFC напрямую (только читает). Нужно одноразовое приложение **Shortcuts**:

1. Открыть Shortcuts → создать automation → «Когда NFC-метка...».
2. Привязать пустую метку.
3. В action добавить «Open URL» с нужным URL.

(Альтернатива — использовать Android-смартфон один раз для записи всех меток.)

## Как использовать

1. Повар подносит телефон к холодильнику → телефон открывает форму заполнения.
2. Заполняет температуру → сохраняет.
3. Готово.

## Ограничения

- **Дальность**: NFC работает 1-3 см, не «через зал». Это хорошо — повар точно знает что считал именно нужный холодильник.
- **iPhone**: на iPhone XS+ есть Background Tag Reading — не нужно открывать никакое приложение, метка срабатывает сама.
- **Android**: на большинстве телефонов NFC по умолчанию выключен. Включить в Settings → Connected devices → NFC.

## Метки, которые мы рекомендуем

| Модель | Память | Цена |
|---|---|---|
| NTAG213 | 144 байт | 2 ₽ |
| NTAG215 | 504 байт | 3 ₽ |
| NTAG216 | 888 байт | 5 ₽ |

URL `/equipment-fill/<id>` ≈ 50 байт — хватает любого варианта. Можно брать самые дешёвые NTAG213.

## Лучше: NFC + QR на одной наклейке

Купить **NFC-наклейки с QR-кодом** (некоторые поставщики предлагают): на лицевой стороне печатается QR (с тем же URL), сзади — NFC-чип. Так:

- В перчатках → NFC.
- Без NFC-телефона → камера + QR.
- Один артикул, один URL, никакого дублирования настройки.

## Когда **не** надо

- На холодильниках, которые часто переставляют — лучше брать перезаписываемые NTAG'и.
- В горячих цехах с >70°C — NFC-чипы могут плавиться. Использовать металлические подложки для термостойкости.

## Поддержка

Сейчас система поддерживает только endpoint `/equipment-fill/[id]`. Для других журналов (например «гигиена сотрудника») потребуется отдельная стратегия — например, NFC-метки на форме каждого работника. Это отдельная задача — пишите в support, обсудим.
